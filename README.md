# Виджет прогресса задач

Скрипт Tampermonkey для тех, кто работает с большим количеством однотипных задач в течении дня. Он считает сколько должно быть сделано и к какому времени.
По умолчанию работает на одном сайте, но в коде можно изменить его на любой другой (7 строчка) // @match        *://любойСайт.com/*
